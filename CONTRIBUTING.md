## How to Contribute

1. **Pull latest commits**: Before making any changes, make sure to pull the latest commits from GitHub:
   ```
   git pull
   ```

2. **Make Your Changes**: Implement your changes in the codebase. Make sure to write clear and concise commit messages.

3. **Commit Your Changes**: Once you are satisfied with your changes, commit them:
   ```
   git commit -m "Description of your changes"
   ```

4. **Push your changes**: Push your changes to the main branch:
   ```
   git push
   ```
